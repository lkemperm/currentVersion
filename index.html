miRcomp Package Introduction

The package uses data from a dilution / mixture experiment to assess methods that estimate microRNA expression from qPCR amplification curves. Specifically, the package provides assessments of accuracy, precision, data quality, titration response, limit of detection, and complete features. Each of these is described in the following sections. To avoid any confusion due to naming conventions (expression estimates from amplification curves have been called Ct values, Crt values, and Cq values to name a few), we refer to the reported values as expression estimates or simply expression. Click here for information about experimental design, assessments and more about how to use the package by itself miRcomp package vignette

Web Application Purpose

There are several achievements of the miRcomp web application. The miRcomp package by itself includes many useful tools for assessment, comparison and visualization but the web application aims to make visualization faster and more dynamic, while adding a few useful tools that accomplish comparisons that can not easily be made using the miRcomp package by itself. The web application is also an easier to use tool for non- R users. All packages required by the application are pre-installed and run smoothly on the server, and it is not necessary to have any version of R installed on your computer at all.

Quality Sliders

One of the most important and useful tools in the miRcomp web application is the quality sliders. The widgets range from 0 to 1, 1 corresponding to filtering all data and 0 corresponding to filtering no data. Below the sliders, a text field reactively displays the corresponding threshold. This threshold (qcThreshold) is a numeric threshold corresponding to a dataset below which values are considered low quality. Quality sliders content, here with pictures
